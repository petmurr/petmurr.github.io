Plotter Art

A 3D printer is really a fancy 2D printer.

To prove it, I modified my 3D printer to draw. Using a 3D printed clamp that uses the print head’s shroud mounts, you can hold a pen that is controllable in 3 axes. I wrote a script in python that converts an SVG image into GCODE, which then can be loaded on the printer. This effectively tells the printer to trace lines, and raise the pen between each line.

I used mitxela’s “plotterfun” website to convert images to SVGs. His website uses a variety of different algorithms to convert regular images to SVGs. I found the linedraw algorithm provides some of the best results for my plotter- some algorithms, like the subline algorithm, produce too many lines. This requires a more precise plotter to recreate accurately- one of my goals for an improved version.
